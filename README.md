# 📊 Amazon Sales Analysis Dashboard (Power BI)

An interactive Business Intelligence dashboard built using Power BI to analyze Amazon sales data and generate actionable insights for business decision-making.

🎓 Developed at Lovely Professional University 🔗  

---

## 🚀 Project Overview
This project focuses on transforming raw sales data into meaningful insights using data visualization and analytics techniques. The dashboard provides a comprehensive view of sales performance, customer trends, and operational metrics.

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query (ETL)
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

---

## 📈 Key Features
- Interactive dashboard with filters (Category, Brand, Payment Method)
- KPI tracking (Orders, Revenue, Tax, Shipping Cost)
- Data modeling for optimized performance
- Advanced DAX calculations for insights
- Clean and user-friendly visualization design

---

## 📊 Insights
- 📦 Total Orders: 33

- 👕 Top Category: Clothing (100%)

---

## 🎯 Business Impact
- Enables real-time performance tracking  
- Identifies top-performing segments  
- Highlights cost drivers (tax & shipping)  
- Supports data-driven decision-making
